  **Key Objectives**
**Data Cleaning:**
    Handled missing values (NaN) and corrected data types for Price, Installs, and Reviews.
**Outlier Detection:**
   Identified and filtered outliers data points (e.g., ratings > 5).
**Profitability Analysis:** 
   Calculated estimated revenue for paid apps to identify top-performing categories.
**Visualization:** 
   Created Bar Charts and Scatter Plots to visualize category distribution and the relationship between user reviews and ratings.

**Technologies**
Python: Pandas, NumPy.
Visualization: Matplotlib, Seaborn.

**Key Insights**
Bar plot show that Family and Games have highest number of apps which indicate high demand.
Scatterplot shows the relationship between rating and reviews.
